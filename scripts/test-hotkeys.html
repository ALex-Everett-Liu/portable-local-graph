Hotkey Mode & Command Palette Test

Command Palette

Press Ctrl+P to open command palette

  • Search for commands by typing
  • Use arrow keys to navigate
  • Press Enter to execute
  • Press Escape to close

Hotkey Mode

Press Alt to activate hotkey mode

  • N - Node Mode
  • E - Edge Mode
  • S - Select Mode
  • F - Search Nodes
  • C - Calculate Centralities
  • L - Manage Layers
  • G - New Graph
  • O - Open Graph
  • Z - Undo
  • R - Redo
  • D - Delete Selected
  • X - Clear Selection
  • I - Zoom In
  • U - Zoom Out
  • 0 - Reset Zoom
  • P - Command Palette
  • ? - Show Help
  • ESC - Exit Hotkey Mode

Global Shortcuts

  • Ctrl+N - New Graph
  • Ctrl+S - Save Graph
  • Ctrl+Shift+S - Save Graph As
  • Ctrl+O - Open Graph
  • Ctrl+Z - Undo
  • Ctrl+Shift+Z - Redo
  • Ctrl++ - Zoom In
  • Ctrl+- - Zoom Out
  • Ctrl+0 - Reset Zoom
  • Delete - Delete Selected
  • Escape - Clear Selection/Exit Mode
